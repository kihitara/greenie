*************************************************
****************** Strawberry *******************
*************************************************

Strawberry is a fixed-width, three column theme
for Moodle 2.

It has a basic settings page with the following
functions:
- Easy setting of theme colour from a list
- Changing the logo
- Adding a footnote
- Removing the link to moodlebites.com in footer
- Custom CSS box
